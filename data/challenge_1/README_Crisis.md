CrisisLexT26 Tweets Dataset (Binary Classification – Competition Use)

Overview
This competition (NeuroLogic 26) uses data derived from the CrisisLexT26 dataset.
We are not the creators or owners of this dataset. It was originally collected and published by the authors cited below.
The dataset contains tweets from 26 real-world crisis events (2012–2013), annotated to support research in crisis informatics and social media analysis.
For this competition, the original labels have been simplified into a binary classification task.
________________________________________
Task Definition
Participants are required to build a model that classifies tweets according to two labels:
1 — Informative
0 — Not Informative
________________________________________
    
The label count: 

Informative 16019

________________________________________
Rationale
•	The goal is to identify actionable, useful information during crisis events.
•	Tweets that are unrelated or lack actionable content are grouped into a single negative class.
•	This formulation reflects real-world use cases such as:
o	Emergency response filtering
o	Situational awareness systems
o	Information prioritization pipelines

Evaluation
•	F1-score
